# ShuttleFrog
HKU 3rd team game project.

A two-player mobile game where one player wields the phone and one wields the controller. Guide the ShuttleFrog through a world of plushies!

## How to Play

ShuttleFrog is a cooperative two-player experience:

- **Player 1** holds the phone and physically rotates it to control the environment.
- **Player 2** uses a Bluetooth controller to move the Shuttle Frog and interact with plushies.

## Requirements

- Android phone
- Standard controller (compatible with Android)
- 2 players
- Some open space to play in

## Installation

1. Download `ShuttleFrog.apk` onto your Android phone.
2. If prompted, allow installation from unknown sources in your device settings.
3. Open the `.apk` file to install the game.
4. Launch and play!
